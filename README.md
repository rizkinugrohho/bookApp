## Getting Started

Follow the steps below to set up the project on your local machine and run it:

1. Clone the repository:

```bash
git clone https://github.com/rizkinugrohho/bookApp.git
cd bookApp
```

2. Backend Setup:

```bash
cd backend
npm install
```

3. Frontend Setup(new terminal):

```
cd frontend
npm install
npm run dev
```

## Technologies Used

### Backend:

- Node.js
- Express.js
- MongoDB

### Frontend:

- Vite
- React
